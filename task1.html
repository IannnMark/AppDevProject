History of Mobile Application Development


The history of the mobile app begins, obviously, with the history of the mobile device and the first mobile phones whose microchips required the basic of software to send and receive calls. But since then things have got a lot more complicated.

Psion EPOC


PSION EPOC, also known as Symbian EPOC, was an early operating system developed by Psion for handheld devices and personal digital assistants (PDAs) in the late 1980s and 1990s. EPOC stands for "Electronic Piece Of Cheese," which was the original codename for the operating system.


Palm OS


Palm OS was a popular operating system developed by Palm, Inc. (formerly known as Palm Computing, Inc.) for personal digital assistants (PDAs) and later, smartphones. It was one of the pioneering operating systems in the handheld computing industry and had a significant impact on the early mobile device market.


WML


Wireless Markup Language (WML) was a markup language used to create content for wireless devices, particularly mobile phones, in the early days of mobile internet. It was designed as a simplified version of HTML to accommodate the limitations and constraints of mobile devices and their slow data connections. WML was widely used in the late 1990s and early 2000s to create mobile-friendly websites and services.


J2ME/JME


Java 2 Micro Edition (J2ME) was a platform developed by Sun Microsystems (later acquired by Oracle Corporation) that allowed developers to create applications for mobile devices and embedded systems using the Java programming language. J2ME was designed to provide a standardized environment for developing Java applications that could run on a wide range of devices with varying capabilities, such as mobile phones, PDAs, set-top boxes, and more.


Symbian


Symbian OS was an operating system designed for smartphones and mobile devices, known for its dominance in the early years of the smartphone industry. It was developed by Symbian Ltd., a consortium of technology companies, and later became the foundation for various Nokia smartphones. Symbian OS offered a versatile platform for application development and played a significant role in shaping the mobile landscape.

Android


Android is a popular and widely used mobile operating system developed by Google. It is designed for smartphones, tablets, smart TVs, and other smart devices. Android has played a significant role in shaping the modern mobile landscape and has become one of the most dominant operating systems globally.



iOS


iOS is a mobile operating system developed by Apple Inc. It is exclusively designed for Apple's mobile devices, including the iPhone, iPad, and iPod Touch. iOS is known for its sleek user interface, strong security features, and tight integration with Apple's ecosystem of hardware, software, and services.



Phone Gap


PhoneGap (now known as Apache Cordova) is an open-source mobile development framework that allows developers to create cross-platform mobile applications using web technologies such as HTML, CSS, and JavaScript. It enables developers to build apps that can run on multiple platforms, including iOS, Android, Windows, and more.



Titanium Mobile


Titanium Mobile is an open-source mobile development framework that enables developers to create cross-platform mobile applications using JavaScript and a single codebase. It allows for the development of native-like apps for various platforms, including iOS, Android, and Windows Phone, by utilizing a combination of JavaScript, APIs, and native components.
